# Установка и запуск BallonsTranslator

## Шаг 1 — Скачайте Python 3.12

1. Перейдите по ссылке:
   https://www.python.org/ftp/python/3.12.9/python-3.12.9-amd64.exe

2. Запустите скачанный файл

3. **ВАЖНО:** на первом экране поставьте галочку **"Add Python to PATH"**

4. Нажмите **Install Now** и дождитесь окончания установки

---

## Шаг 2 — Скачайте Git

1. Перейдите по ссылке:
   https://git-scm.com/download/win

2. Скачайте и установите (все настройки оставьте по умолчанию)

---

## Шаг 3 — Скачайте проект

1. Откройте **Пуск** → найдите **"Командная строка"** → запустите

2. Введите команду и нажмите Enter:
```
git clone https://github.com/Samudinov00/manga-ocr.git
```

3. Перейдите в папку проекта:
```
cd manga-ocr
```

---

## Шаг 4 — Установка

Дважды кликните на файл **`setup.bat`**

Установка займёт **5–15 минут** (скачиваются библиотеки).
Дождитесь сообщения **"Установка завершена успешно!"**

---

## Шаг 5 — Запуск

Дважды кликните на файл **`run.bat`**

---

## Возможные проблемы

**"Python не найден"**
→ При установке Python не поставили галочку "Add Python to PATH"
→ Переустановите Python, не забудьте про галочку

**Программа запустилась но не переводит**
→ Нужен API ключ переводчика (DeepL, ChatGPT)
→ Или используйте бесплатный **"DeepL Free"** или **"google"** — ключ не нужен

---

## Системные требования

- Windows 10 / 11 (64-bit)
- Видеокарта NVIDIA (рекомендуется, но не обязательно)
- 8 ГБ RAM и более
- 10 ГБ свободного места на диске
